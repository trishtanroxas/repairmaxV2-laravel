🎯 API ENDPOINTS

Authenticated (Require Bearer Token):
  POST   /api/chatbot/message - Send message
  GET    /api/chatbot/sessions - List conversations
  GET    /api/chatbot/sessions/{id}/messages - Get messages
  POST   /api/chatbot/sessions - Create session
  DELETE /api/chatbot/sessions/{id} - Delete session

Webhooks (From N8N - No Auth):
  POST /api/webhooks/n8n/chatbot-response
  POST /api/webhooks/n8n/repair-status-update
  POST /api/webhooks/n8n/booking-confirmation
  POST /api/webhooks/n8n/appointment-notification
  POST /api/webhooks/n8n/product-recommendation
  POST /api/webhooks/n8n/support-ticket

🌐 DEPLOYMENT TO PRODUCTION

1. Create Cloudflare Tunnel:
   • Go to https://dash.cloudflare.com/tunnels
   • Create tunnel "repairmax-n8n"
   • Get tunnel token

2. Configure Environment:
   $ echo "CLOUDFLARE_TUNNEL_TOKEN=eyJhIjoiXX..." >> .env

3. Restart Services:
   $ docker-compose down
   $ docker-compose up -d

4. Create Public Hostname:
   • Subdomain: repairmax
   • Domain: n8n.app
   • URL: http://n8n-app:5678

5. Access Production:
   https://repairmax.n8n.app

🆘 QUICK TROUBLESHOOTING

Problem: Can't access http://localhost:5678
→ Solution: docker-compose restart n8n

Problem: Database tables missing
→ Solution: php artisan migrate

Problem: N8N can't reach Laravel
→ Solution: Use http://host.docker.internal:8000 in Docker

Problem: Cloudflare tunnel not connecting
→ Solution: Check CLOUDFLARE_TUNNEL_TOKEN in .env
